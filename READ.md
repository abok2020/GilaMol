# GilaMol

GilaMol is a computational platform for molecular discovery and design.

The project aims to integrate:

- cheminformatics workflows
- molecular docking
- graph neural network (GNN) property prediction
- autonomous AI agents for molecular design

## Planned Components

1. **Molecular Processing**
   - SMILES parsing
   - descriptor calculation
   - drug-likeness filtering

2. **Docking Pipeline**
   - AutoDock Vina integration
   - receptor–ligand docking
   - binding score extraction

3. **Property Prediction**
   - graph neural network models
   - ADMET prediction

4. **Molecule Ranking**
   - multi-objective scoring
   - candidate prioritization

5. **Agent Layer**
   - AI-driven workflow orchestration
   - automated design loops
